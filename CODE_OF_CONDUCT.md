# 贡献者行为准则 · Contributor Covenant

## 我们的承诺

作为项目的成员、贡献者与维护者，我们承诺让参与本社区的每一位成员，都能享有免于骚扰的体验，不因年龄、体型、身心障碍、族裔、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人形貌、种族、宗教信仰、性取向而受到区别对待。

我们承诺以能促进开放、友好、多元、包容、健康社群的方式行事、交流与互动。

## 我们的准则

有助于社区正向氛围的行为包括：

- 展现同理心与善意
- 尊重不同的观点、意见与经验
- 给予和接受建设性的反馈
- 承担责任，向受影响的个人道歉，并从经验中学习
- 关注对整体社区最有利的事情，而不只是个人

无法接受的行为包括：

- 使用带有性暗示的语言或图像，以及性方面的关注或求爱
- 挑衅、侮辱、贬低性言论，以及针对个人或政治的攻击
- 公开或私下骚扰
- 未经明确许可，公开他人的私人信息，例如实际或电子地址
- 其他在专业场合中被合理认定为不当的行为

## 执行责任

社区维护者有责任说明并执行我们的可接受行为准则，并对任何被视为不当、威胁、冒犯或有害的行为，采取适当且公平的纠正措施。

社区维护者有权利与责任移除、编辑、拒绝违反本准则的评论、提交、代码、维基编辑、Issue 与其他贡献，并在适当时说明理由。

## 适用范围

本行为准则适用于所有社区场域，也适用于个人在公开场合代表社区的场合（例如使用官方电子邮件地址、社交媒体官方账号发文，或作为指定代表参与线上或线下活动）。

## 举报与执法

若发生辱骂、骚扰或其他不可接受的行为，请通过 GitHub 项目 Issue（标签 `code-of-conduct`）或 [Security Advisory](https://github.com/gandli/vmware-downloads/security/advisories/new) 私下举报。所有投诉都将得到及时且公正的审查与调查。

社区维护者有义务尊重举报人的隐私与安全。

## 执行准则

社区维护者将依据以下社区影响准则，判断违反本行为准则应采取的行动：

### 1. 更正

**社区影响**：使用不当语言或其他被认为在社区中不专业或不受欢迎的行为。

**后果**：由社区维护者私下、书面警示，说明违反行为的性质与不当理由。可能要求公开道歉。

### 2. 警告

**社区影响**：单一事件或一连串行为构成违规。

**后果**：警告并说明后续行为的影响。在指定期间内，不得与相关人员产生任何互动，包括主动接触执法准则的执行者。违反此条款可能导致临时或永久性封锁。

### 3. 临时封锁

**社区影响**：严重违反社区规范，包括持续的不当行为。

**后果**：在指定期间内，禁止以任何形式与社区互动或公开沟通。违反此条款可能导致永久性封锁。

### 4. 永久封锁

**社区影响**：表现出持续违反社区规范的模式，包括持续的不当行为、骚扰个人，或对特定群体的攻击或贬损。

**后果**：永久禁止在社区内公开互动。
